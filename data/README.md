## Adding a new data source

1. Verify the license is permissive (MIT, Apache 2.0, BSD, CC-BY,
   public domain, or self-owned). **Reject CC-BY-SA, GPL, AGPL,
   LGPL.** See ADR 0002.
2. Add the file under the appropriate subfolder.
3. Document the source, license, and attribution in this README.
4. If the data needs processing, add an ingestion script in
   `scripts/`.
5. If schema changes are required, write an ADR first.
