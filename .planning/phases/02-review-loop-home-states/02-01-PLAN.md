    - dayKeyLocal(date): returns a stable local-day key (e.g. 'YYYY-MM-DD' from local year/month/date), so two timestamps on the same local calendar day map to the same key. D-02: device-local midnight boundary.
    - localDayStart(now): returns a Date at 00:00:00.000 local time for `now`.
    - computeStreak(reviewedAtDates: Date[], now: Date): number.
        - D-01: a day is "active" if >=1 review occurred that local day.
        - Counts consecutive active days ending today OR yesterday (streak stays alive on a day with no reviews yet only if yesterday was active — alive but not yet incremented today). Returns the run length of consecutive active days ending at the most recent active day, provided that most recent active day is today or yesterday; otherwise 0 (broken).
        - Empty input -> 0.
        - Test cases: reviews today+yesterday+2-days-ago -> 3; reviews yesterday+2-days-ago but none today -> 2 (alive, at-risk); reviews only 3 days ago -> 0 (broken); multiple reviews same day -> counts that day once; empty -> 0.
    - computeTodayStats(reviews: {reviewed_at: Date; rating: string}[], now: Date): { reviewed: number; again: number }.
        - reviewed = count of reviews whose reviewed_at is the same local day as now.
        - again = count of those with rating === 'again'.
        - D-11 accuracy is derived downstream as (reviewed - again)/reviewed; this fn only returns the two counts.
        - Test cases: mix of today/yesterday reviews -> only-today counted; again count correct; empty -> {reviewed:0, again:0}.
  </behavior>
