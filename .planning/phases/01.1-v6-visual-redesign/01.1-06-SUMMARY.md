# 01.1-06 — Streak Detail Reskin (v6)

## What was built

- **Task 1 — `src/components/Heatmap.tsx` (commit `55442d8`):** `YearHeatmap` renders a
  53-week × 7-day grid built column-by-column from React Native `View`s (53 flex columns, each
  a stack of 7 `aspectRatio:1` cells, gap 2, radius 1.5), mapping intensity 0–3 to
  `colors.heat0..heat3`. `todayIndex` renders transparent with a 1px brand outline (not filled).
  The section header carries the "less ▢▢▢▢ more" ramp legend, and a flex-spaced month axis sits
  underneath. `MonthHeatmap` renders a 7-column calendar grid with an `M T W T F S S` Mono
  header, transparent leading-empty days, and today outlined 1.5px brand. All colors from theme.
- **Task 2 — `src/components/RatingDistribution.tsx` (commit `e34cf4e`):** a 14px stacked
  horizontal bar (radius 4, overflow hidden) split into `again/hard/good/easy` flex segments in
  the semantic tokens, plus a 4-column legend (swatch + uppercase Mono name + bold mono percent).
  Accepts either `counts` or `percents` (+ optional `total`); zero-state renders an empty
  `paperSoft2` bar and 0% legend. Wrapped in a paper `Card` with a "{total} reviews" header.
- **Task 3 — `app/streak/index.tsx` (commit `0e3c6c0`):** replaced the V0 static stub with the
  full v6 layout in a scrollable light surface — detail nav (← `router.back()`, centered title,
  ↗ share placeholder); compact hero (Ionicons `flame` in accent gold + cobalt `Num` 44 + Mono
  caps line `day streak · longest N · retention N%`); a working `period-toggle` (month / year /
  lifetime) defaulting to `year` with the active segment as a shadowed paper pill; a 2×2
  `StatTile` grid; period-scoped activity (`YearHeatmap` / `MonthHeatmap` / lifetime monthly
  bars); a reviews bar chart (brand-fill, last bar highlighted); the `RatingDistribution`; and a
  personal-bests `Card` list with the current run on a brand-tint row. The toggle re-scopes the
  stat grid, activity layout, reviews chart, and rating distribution.
