<interfaces>
hono/jwk (ES256 path): `jwk({ jwks_uri, alg: ['ES256'] })` — verifies against JWKS endpoint, stores
payload in context under key 'jwtPayload', rejects algorithms not in the allowlist.
hono/jwt (HS256 path): `jwt({ secret, alg: 'HS256' })` — verifies with shared secret.
Both throw/respond 401 on invalid or missing token. Read payload via `c.get('jwtPayload')`.
Hono version 4.12.27 is already a dependency (no install).
</interfaces>
